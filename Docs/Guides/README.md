## 🔍 指南类型说明

### 操作指南 (How-to Guides)
- 针对具体任务的步骤说明
- 侧重实践操作
- 包含代码示例

### 最佳实践 (Best Practices)
- 经验总结和建议
- 提供多种方案对比
- 说明优缺点

### 教程 (Tutorials)
- 系统性学习文档
- 从简单到复杂
- 包含练习和验证
