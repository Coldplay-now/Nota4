## 🎯 核心功能

### Mermaid 图表

- **图表渲染**：支持流程图、时序图、甘特图等
- **主题支持**：支持明暗主题切换
- **TOC 兼容**：与目录功能兼容
- **错误处理**：渲染失败时的错误处理
