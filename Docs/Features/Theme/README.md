## 🎯 核心功能

### 主题系统

- **主题管理**：支持多主题切换
- **系统集成**：与 macOS 系统主题集成
- **自定义主题**：支持用户自定义主题
- **主题持久化**：主题设置保存和恢复

### 代码高亮

- **语法高亮**：支持多种编程语言
- **主题切换**：代码高亮主题选择
- **库选择**：Ink/Splash/Highlight.js 等

### Mermaid 主题

- **图表主题**：Mermaid 图表主题支持
- **暗色模式**：暗色主题可见性优化
